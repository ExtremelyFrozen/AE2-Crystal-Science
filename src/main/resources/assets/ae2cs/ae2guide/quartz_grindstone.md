# 石英磨具

<Row>
  <BlockImage id="quartz_grindstone" scale="8" />
</Row>

石英磨具是游戏早期用于获取水晶粉尘的一种方式，它可以是电驱动或曲柄驱动的，不能插入加速卡，主要用于游戏早期的过渡阶段。

机器特性：支持[面配置](side_config.md)功能、支持通过外部线缆获取能量、不能传递频道
