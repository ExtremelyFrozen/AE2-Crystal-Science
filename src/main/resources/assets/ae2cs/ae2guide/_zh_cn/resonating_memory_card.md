# 谐振内存卡

<Row gap="16">
  <ItemImage id="ae2cs:resonating_memory_card" scale="4" />
</Row>

**谐振内存卡** 是 AECS 对 AE 内存卡的扩展版本。
它保留了内存卡的基本配置复制能力，
并提供多个独立槽位来保存不同机器或面板的设置。

---

## 多槽位存储

谐振内存卡一共拥有 **9 个独立槽位**。

- 每个槽位都可以保存一套独立配置
- 各槽位之间互不影响
- 当前选中的槽位会在 tooltip 中显示其编号与保存的配置名称

这使得一张内存卡就可以同时管理多台设备的常用设置。

---

## 槽位切换与清空

谐振内存卡支持快捷切换槽位：

- **Shift + 滚轮** 切换当前槽位
- 切换时会提示当前槽位编号，以及该槽位中保存的配置名称
- **Shift + 右键** 可清空当前槽位的数据

如果某个槽位没有保存内容，
它会显示为空白配置。

---

## 自动应用配置

谐振内存卡最重要的扩展功能是**自动应用**：

- 当玩家背包中带有一张当前槽位已保存数据的谐振内存卡时
- 新放置的 AE 方块或 AE 面板会自动尝试导入该槽位中的配置

自动应用成功时，
会提示玩家当前使用的是哪一个槽位以及其中的配置名称。

这可以显著减少批量布置设备时的重复配置操作。

---

## 使用建议

一个常见做法是：

- 为不同机器类型分别预留不同槽位
- 或为同一种机器准备多套不同工作模式

例如可以把一组槽位专门留给：

- 频段设备
- 样板供应器
- 面配置类机器

这样在扩建网络时，
只需要切换槽位并继续放置设备即可。
