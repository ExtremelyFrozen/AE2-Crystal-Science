# 熵变反应仓

<Row gap="16">
  <BlockImage id="ae2cs:entropy_variation_reaction_chamber" scale="8" />
</Row>

**熵变反应仓** 是 AECS 提供的一种特殊机器，
用于在机器内部执行 **<ItemLink id="ae2:entropy_manipulator" />** 的相关配方。

---

## 机器特性

熵变反应仓具备以下通用机器特性：

- 支持 **[面配置](side_config.md)** 系统
- 可自动从所连接的 **ME 网络** 中获取能量
- 支持通过外部能量线缆获取能量
- **参与频道传递**
