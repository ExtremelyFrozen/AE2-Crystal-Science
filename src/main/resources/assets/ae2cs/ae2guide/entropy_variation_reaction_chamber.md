# Entropy Variation Reaction Chamber

<Row gap="16">
  <BlockImage id="ae2cs:entropy_variation_reaction_chamber" scale="8" />
</Row>

The **Entropy Variation Reaction Chamber** is a specialized machine provided by AECS.
It is designed to perform recipes related to the **<ItemLink id="ae2:entropy_manipulator" />** entirely within the machine.

---

## Machine Characteristics

The Entropy Variation Reaction Chamber has the following general machine properties:

- Supports the **[Side Configuration](side_config.md)** system
- Can automatically draw power from the connected **ME network**
- Can also receive power via external energy cables
- **Participates in channel transmission**
