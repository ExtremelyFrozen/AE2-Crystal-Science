# 熵变反应仓

## 熵变反应仓

<Row>
  <BlockImage id="entropy_variation_reaction_chamber" scale="4" />
</Row>

熵变反应仓可以让你在机器内做到原本只有熵变机械臂才能进行的操作。如果你和我一样是一个不喜欢刷附魔的玩家，那么这会是一个十分简单的获取冰的机器。

机器特性：支持[面配置](side_config.md)功能、自动从AE网络中获取能量、支持通过外部线缆获取能量、可以传递频道。
