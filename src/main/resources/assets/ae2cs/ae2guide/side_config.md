# 机器的面配置

## 机器的面配置
![面配置](assets/side_config.png)

如图所示，AECS的机器使用一套容易操作且实用的面配置系统。你可以指定输入输出面，也可以让机器自动弹出产物并自动抽入原料。
在面配置中：红色表示输入、蓝色表示输出、紫色表示既可以输入也可以输出，而灰色表示这个面不允许输入输出
无论选用哪一种，机器不会把自己的输入槽位输出，也不会把原料抽入输出槽位中。
侧面三个按钮：分别标志着主动输入、主动弹出、清空面配置。当主动模式启用时，对应的按钮会保持高亮状态
当你打开主动输入时，机器只会从允许输入的面抽取资源；
主动弹出也只会从允许输出的面弹出
