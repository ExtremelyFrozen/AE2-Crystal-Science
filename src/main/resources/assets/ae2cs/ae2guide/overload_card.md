# Meteorite Overload Card

<Row gap="16">
  <ItemImage id="ae2cs:overload_card" scale="4" />
</Row>

The **Meteorite Overload Card** is a high-tier upgrade card for AECS processing devices.

---

## Meteorite Pattern Provider

When installed in the **<ItemLink id="ae2cs:meteorite_pattern_provider" />**:

- Each card provides **256 parallel crafts**
- Up to **4 cards** can be installed

This makes it suitable for very high-throughput internal crafting.

---

## AECS Processing Machines

When installed in supported AECS processing machines:

- The machine is forced to its fastest processing state
- A craft finishes in **4 ticks**
- Only **1 card** can be installed per machine

This behavior is intended for recipe-processing machines rather than general utility devices.
