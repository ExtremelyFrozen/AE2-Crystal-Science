# 晶能聚合器

## 晶能聚合器

<Row>
  <BlockImage id="crystal_aggregator" scale="4" />
</Row>

模组中绝大部分物品都需要使用这台机器制作。同时，其可以以同样的材料制作更多地水晶种子，因此也是增产高级水晶的必备工具。

机器特性：支持[面配置](side_config.md)功能、自动从AE网络中获取能量、支持通过外部线缆获取能量、可以传递频道。
