# 空白样板

<Row gap="16">
  <ItemImage id="ae2cs:blank_print_press" scale="2" />
  <ItemImage id="ae2cs:ender_blank_print_press" scale="2" />
</Row>

**空白样板** 是 AECS 提供的一种用于样板本身存储的工具。
它可以通过样板合成，也可以在切石机中重新转换为各类样板。

---

## 样板加工

空白样板可以在**切石机**中进行加工，
被制造为不同类型的样板。
