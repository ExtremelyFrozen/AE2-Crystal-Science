# 水晶工具

## 工具——高纯水晶最简单直接的用途
AECS并没有提供一套过于强大的工具，它们仍然在原版工具的范畴内，不过它们仍然好用，尤其是相对于其低廉的造价。

末影水晶工具拥有钻石等级的挖掘能力、速度和耐久，并且可以绑定到<ItemLink id="ae2:wireless_access_point" />，任何通过末影水晶工具挖掘的方块或击杀得到生物的掉落物都会直接进入其绑定的ME网络库存。
此外，与终端不同的是，末影水晶工具的连接能力不需要其在无线接入点附近也能生效，只要其绑定的无线接入点处于被加载的区块即可。

谐振水晶工具是陨石水晶工具和末影水晶工具的结合与升级。它不仅具备末影水晶工具的掉落物传送能力，而且拥有下界合金级别的挖掘能力、金工具的挖掘速度以及陨石水晶工具2倍的耐久度。
每一件谐振水晶工具都自带附魔：
剑、斧：锋利Ⅲ
镐：时运Ⅲ
锹：精准采集
锄：效率Ⅴ
