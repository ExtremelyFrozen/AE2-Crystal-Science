# 赛特斯石英矿石

AECS不仅带回了高纯赛特斯石英水晶，而且同时带回了赛特斯石英矿石以及深层版本。

## 石英矿石
<Row>
  <BlockImage id="certus_quartz_ore" scale="4" />
  <BlockImage id="deepslate_certus_quartz_ore" scale="4" />
  <BlockImage id="charged_certus_quartz_ore" scale="4" />
  <BlockImage id="deepslate_charged_certus_quartz_ore" scale="4" />
</Row>

这些矿石会在-59层到+72层之间均匀分布。其中，充能版本约占生成量十分之一，并且自身发出亮度为7的光照。
木镐即可挖掘这些矿石，默认掉落1到2个，如果工具附魔时运3，则可掉落1到8个。
精准采集可以直接挖掘此矿石，矿石可以被晶能粉碎机粉碎，产出6个赛特斯石英水晶（即使是充能矿石，直接粉碎后也不会得到充能水晶）。
