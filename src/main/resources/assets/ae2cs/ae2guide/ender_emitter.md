# 末影发信器

<Row>
  <ItemImage id="ender_emitter" scale="8" />
  <ItemImage id="ender_linker" scale="8" />
</Row>

末影发信器是适用于短距离的全自动无线链接装置。只需将其接上线缆，它便会在一定范围内自动链接新放置的方块，直到自身频道全部用完。
可以配合<ItemLink id="ender_broadcaster" />建立一个完全无AE线缆的基地。

如果基地内需要使用子网而不想被其影响，你可以选择关闭对线缆和线缆部件的自动链接，或者临时关闭自动链接模式。

此外，即使完全关闭自动链接模式，你也可以使用<ItemLink id="ender_linker" />手动设置链接，通过shift右键绑定发信器，右键链接即可。

如果你想更精细的控制自动链接，你可以设置自动链接的范围，这样，发信器只会链接自动范围内的设备，对于超出了自动范围而不超过最大范围的设备，仍然可以手动链接

多个末影发信器可以互相链接，进行远距离频道传输，不过一般来说<ItemLink id="ender_broadcaster" />更能胜任远距离传输任务。
