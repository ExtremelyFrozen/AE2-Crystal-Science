# 共振回路

<Row gap="16">
  <ItemImage id="ae2cs:resonating_print_press" scale="2" />
  <ItemImage id="ae2cs:resonating_circuit_print" scale="2" />

**共振回路**は、AECSで高度な機械やネットワークコンポーネントを
クラフトするために使用される中核素材の一つです。
AECSの進捗において高ティアの回路として位置づけられ、
主に共振水晶に関連する機能デバイスの構築に使用されます。

---

## 構成部品とクラフト工程

共振回路の生産は以下の手順で構成されます：

1. **共振回路の金型**をクラフトする
2. インスクライバーを使用して**共振回路プリント**を製作する
3. 回路プリントをさらに加工して**共振回路**を完成させる
