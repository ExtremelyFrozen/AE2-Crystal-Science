# 电路蚀刻器

## 电路蚀刻器

<Row>
  <BlockImage id="circuit_etcher" scale="4" />
</Row>

电路蚀刻器是一种高效的处理器获取方式，其提供了直接块状方块到处理器的转变，省去了使用压印模板压制电路板和硅板的过程。

机器特性：支持[面配置](side_config.md)功能、自动从AE网络中获取能量、支持通过外部线缆获取能量、可以传递频道。
